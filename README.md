A Leo program example code

# aleolottery.aleo

## Build Guide

To compile this Aleo program, run:
```bash
aleo build
```

## Run Guide

To run this program, run:
```bash
leo run play <inputs>

or 

./run.sh
```

## Execute Guide

To execute this program, run:
```bash
leo execute play <inputs>
```
